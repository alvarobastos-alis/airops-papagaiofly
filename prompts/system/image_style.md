# Papagaio Fly — Image Style Guide

<!-- Última atualização: 2026-04-25 -->

---

## Identidade Visual

Todas as imagens e assets visuais gerados devem seguir a identidade visual da **Papagaio Fly**.

---

## Paleta de Cores

| Nome | Hex | Uso | Sensação |
|------|-----|-----|----------|
| **Primary** | `#1E3A5F` | Deep Navy Blue | Confiança, profissionalismo |
| **Secondary** | `#F5A623` | Warm Amber | Energia, acolhimento, tropicalidade |
| **Accent** | `#00B4D8` | Sky Blue | Aviação, liberdade, céu |
| **Success** | `#10B981` | Emerald Green | Confirmação, voo no horário |
| **Warning** | `#F59E0B` | Amber | Atenção, atrasos |
| **Danger** | `#EF4444` | Red | Cancelamento, erro |
| **Neutral BG** | `#F0F0F0` | Light Gray | Fundos, espaçamentos |
| **Dark BG** | `#0F172A` | Slate 900 | Dashboard, dark mode |
| **Text Dark** | `#1A1A2E` | Quase preto | Texto principal |
| **Text Light** | `#FFFFFF` | Branco | Texto sobre fundos escuros |

### Gradientes da Marca
- **Hero gradient**: `linear-gradient(135deg, #1E3A5F 0%, #00B4D8 100%)`
- **Warm gradient**: `linear-gradient(135deg, #F5A623 0%, #EF4444 100%)`
- **Sky gradient**: `linear-gradient(180deg, #00B4D8 0%, #F0F0F0 100%)`

---

## Estilo Visual

- Design **clean e moderno** — minimalismo com propósito
- Linhas geométricas suaves e cantos arredondados (8px — 16px)
- Gradientes sutis (navy → sky blue) para profundidade
- Ícones flat com bordas arredondadas
- Fotografias com tratamento de cor fria (azul/cinza)
- Ilustrações vetoriais minimalistas
- Sombras suaves (0 4px 6px rgba(0,0,0,0.1))
- Glassmorphism para cards premium (backdrop-blur + borda translúcida)

## Elementos da Marca

### Mascote/Logo
- **Papagaio** estilizado — colorido, tropical, amigável
- Variações: logo completo (horizontal), ícone (apenas papagaio), favicon

### Elementos Visuais
- ✈️ Aviões estilizados em ângulo ascendente (otimismo, viagem)
- ☁️ Nuvens abstratas como elementos decorativos
- 🌍 Globo terrestre para contexto internacional
- 🎫 Passaporte/bilhete como ícones de viagem
- 🦜 Papagaio como elemento de identidade da marca

---

## Restrições de Imagem

1. **Nunca** usar imagens de pessoas reais identificáveis
2. **Nunca** usar logotipos de companhias concorrentes
3. **Nunca** usar imagens que sugiram acidente, crash ou perigo
4. **Nunca** usar imagens com conotação política, religiosa ou sexual
5. Manter contraste acessível (WCAG AA mínimo — 4.5:1 para texto)
6. Fundo transparente para ícones e logos (PNG)
7. Assets em SVG quando possível para escalabilidade
8. Imagens de aeronaves devem ser genéricas (sem pintura de outras cias)
